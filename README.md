# 安装及启动
## 1. 安装依赖 
`npm install`
## 2. 启动项目(请保证nodejs的版本在7.0以上)
`npm start`
## 3. 运行网页
### 浏览器输入 `localhost:8080`
## 3. 打包项目
`npm run build`
